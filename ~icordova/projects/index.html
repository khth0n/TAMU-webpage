Autodrive Challenge

My current contribution to the TAMU Autodrive Challenge II group involves integrating the pre-existing code base used for the car's processing to actually run on the physical car using. Additionally, through this project, I am also learning substantial amounts of information regarding image processing for object detection, recognition, and tracking, combining it with LiDAR data processing and information. Finally, I am also learning about using a CAN bus to transmit and receive signals both to a processing center and among microcontrollers and similar low-level electronics in order to create the car's control system.
